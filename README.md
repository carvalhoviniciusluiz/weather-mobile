## Sobre o projeto <a name="about"></a>

### **Requisitos:**

- [NodeJs `>17.0.0`](https://nodejs.org/en/)

### **Instalação:**

No terminal faça:

```bash
yarn
```

### **Rodando o Projeto:** <a name="run"></a>

Execute o projeto usando o comando:

```bash
yarn ios

# para escolher um emulador especifico
# yarn ios --simulator="iPhone 14"
```
